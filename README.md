# Cafe Sales Dashboard (Excel Project)

# Objective

To analyze and visualize a café’s sales data and build an interactive Excel dashboard that helps track overall business performance, identify sales trends, and monitor payment and location distribution.

# Tools Used

Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Data Cleaning (Power Query / formulas)

# Dashboard Insights

# Key Metrics:

Total Transactions: 9,439
Total Quantity Sold: 29,343
Total Revenue: ₹84,290.5

# Item-wise Performance:

Top-selling items: Salad, Sandwich, Smoothie.

# Month-wise Trend:

Peak sales observed in June and October.

# Location Insights:

40% of sales are uncategorized, 30% in-store, 30% takeaway.

# Payment Methods:

Cash (22%), Credit Card (23%), Digital Wallet (23%), Undefined (32%).

# Day-wise Sales:

Monday and Friday show higher transaction volumes.

# Key Learnings

Built end-to-end Excel dashboard using raw sales data.
Learned KPI selection and data visualization principles.
Practiced using Excel slicers for interactivity.
Understood how to communicate insights visually for decision-making.
